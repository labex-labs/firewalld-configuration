Firewalld 구성을 학습합니다. 이 모듈은 Firewalld 기본 사항 및 존 (Zones), 서비스 및 포트 관리, 고급 규칙 (Rich Rules), 포트 포워딩 (Port Forwarding), 비상 모드 (Panic Mode) 를 다룹니다. 실습 랩과 실제 환경 기반의 과제를 통해 이러한 필수적인 Linux 기술을 숙달하게 될 것입니다.
