学习 Firewalld 配置。本模块涵盖 Firewalld 基础和区域 (Zones)，管理服务和端口 (Services and Ports)，富规则 (Rich Rules)，端口转发 (Port Forwarding)，以及紧急模式 (Panic Mode)。您将通过实践实验和真实世界的挑战来掌握这些基本的 Linux 技能。
